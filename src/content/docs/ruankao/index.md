### 任意进制转换十进制

```
二进制:10010010.110 -> `1*2^7+1*2^4+1*2^1+1*2^-1+1*2^-2=146.75`
```

### 十进制转换任意进制

```
94转换二进制：
1. 94 / 2 = 47 余 0
2. 47 / 2 = 23 余 1
3. 23 / 2 = 11 余 1
4. 11 / 2 = 5 余 1
5. 5 / 2 = 2 余 1
6. 2 / 2 = 1 余 0
结果：1011110
```

## 机器数

### 符号位

- `+`，符号位为0
- `-`，符号位为1

### 原码

符号位遵循`0正1负`，其他不变

### 反码

- 若符号位为0，数值位不变
- 若符号位为1，数值位全部取反

```
+19D
原 0 0010011
反 0 0010011

-19D
原 1 0010011
反 1 1101100

+0
原 0 0000000
反 0 0000000

-0
原 1 0000000
反 1 1111111
```

### 补码

- 正数的补码=原码
- 负数的补码=反码末位+1

```
+19D
原 0 0010011
反 0 0010011
补 0 0010011

-19D
原 1 0010011
反 1 1101100
补 1 1101101

+0
原 0 0000000
反 0 0000000
补 0 0000000

-0
原 1 0000000
反 1 1111111
补 0 0000000
```

### 移码

在补码的基础上将符号位取反。注意：移码只能用于表示整数。

```
+19D
原 0 0010011
补 0 0010011
移 1 0010011

-19D
原 1 0010011
补 1 1101101
移 0 1101101
```
