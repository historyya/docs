## 常用命令

### 查看发行版

```shell
$ cat /etc/os-release
# 或
$ cat /etc/issue
Debian GNU/Linux
```

### 查看内核版本

```shell
# -a：显示全部的信息，包括内核名称、主机名、操作系统版本、处理器类型和硬件架构等
$ uname -a

# -r:显示内核版本号
$ uname -r

# 查看内核参数
$ cat /proc/version
```

### 查看内存

```shell
# 查看内存的详细信息，包括总内存、剩余内存、可使用内存等
$ cat /proc/meminfo

# 查看系统总内存大小
$ free -h
```

### 查看CPU信息

```shell
# 查看CPU的详细信息，包括CPU型号、类型以及大小
$ cat /proc/cpuinfo

# 查看每个物理CPU含有的核心个数
$ cat /proc/cpuinfo | grep "cpu cores" | uniq
```

### 查看IP

```shell
# 显示系统中所有可用网卡的相关网络信息
$ ip -a

# 查看某块网卡的信息
$ ip addr show eth0

# 查看服务器公共IP地址
$ ip addr show eth0 | grep inet | awk '{ print $2; }' | sed 's/\/.*$//’
```

### 查看路由表

```shell
# 查看路由信息
$ ip route show

# 获取特定IP的路由
$ ip route get 192.168.0.1
```

### 切换用户

```shell
# 切换到root
$ sudo -i
```

### 备份

```shell
# 备份appdata目录
$ cp -r appdata/ appdata.bak
```

## Debian

```bash
# 更新索引
$ sudo apt update

# 安装vim
$ sudo apt install vim
```
